# AI-Procrastination-Predictor
An AI-driven system that proactively predicts when a user is likely to procrastinate and suggests the best study time based on personal app usage behavior.


## Overview
This project predicts procrastination risk before it happens by analyzing app usage behavior.
It also recommends the best study time based on personal focus patterns.

## Problem Statement
Many productivity apps only show screen time after it is already wasted.
This project aims to proactively predict when a user is likely to procrastinate and help them act early.

## What This Project Does
- Analyzes app usage patterns
- Predicts procrastination risk
- Suggests optimal study time windows

## Target Users
Students who struggle with focus and procrastination.
The same approach can be extended to working professionals.

## Project Status
This project is currently under development.
